CSS3 Selector

Attribute Selector［속성 선택자］

class 속성이 「〜로 시작된다」요소를 선택

  • class="STARTxx"
  • class="xxSTART"
  • class="xxSTARTxx"

class 속성이 「〜로 끝난다」요소를 선택

  • class="ENDxx"
  • class="xxEND"
  • class="xxENDxx"

class 속성이 「〜를 포함한다」요소를 선택

  • class="CNTxx"
  • class="xxCNT"
  • class="xxCNTxx"

링크 종류별로 아이콘을 표시

  • 일반 링크
  • 외부 사이트로의 링크
  • PDF 파일 링크

Pseudo Class Selector［유사 클래스］

모든 자식 요소를 카운트하는「〜child」계 유사 클래스

  • child1 (first)
  • child2
  • child3
  • child4
  • child5
  • child6
  • child7 (last)

같은 요소만 카운트하는「〜of-type」계 유사 클래스

heading1 (h4)

paragraph1

heading2 (h4)

paragraph2

heading3 (h5)

paragraph3
